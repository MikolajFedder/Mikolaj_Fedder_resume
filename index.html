A nike database which acts as a storage system for all Nike shoes. Each Nike shoe contains the following details:

  • The 'code' of the shoe.
  • The 'product', the shoe name itself.
  • The 'cost' which it takes to make the shoe.
  • The 'quantitiy', which resembles how much of that shoe is in stock.

The user who is operating the database will have the following options (which will be explained later on):

  • Capture shoes.
  • View all.
  • Re stock.
  • Search shoe.
  • Value per item.
  • Highest quantitiy item.
